Dimmed Theme for Sublime Text 2

Тема для редактора Sublime Text 2 основанная на теме Tomorrow, с исправленными цветами, и более мягкой палитрой.

Скриншоты

Установка

Если вы используете отличный плагин Sublime Package Control от Will Bond, вы можете легко установить его через меню Package Control: Install Package и указав Dimmed Color Scheme Тему можно будет найти в меню Color Scheme Или скачать ZIP архив и скопировать в папку Sublime Text 2\Data\Packages\User\
